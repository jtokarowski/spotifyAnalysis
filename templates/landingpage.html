Music in Context

Discover Music in Context

A project by John Tokarowski

This tool will ask you for Spotify Authentication

The current iteration of the model utilizes k-means clustering to detect sub-genres in a user's spotify playlists.
Select 1 or more playlists to analyze, and the model will create up to 5 playlists from all associated songs.
Spotify recommendations for each sub-genre playlist will be more precise.

For more information on this project, check out my github or blog
